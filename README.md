# G-code Studio — Bambu Lab A1 mini / eSun PLA+

ブラウザだけで動く、**G-code を直接いじれる**スライサー／エディタです。外部通信ゼロ・単一 HTML。
`index.html` をブラウザで開くだけで使えます（サーバー不要・オフライン動作）。

## 特長

- **A1 mini + eSun PLA+ プロファイル内蔵**（ノズル/ベッド温度・フロー比・最大体積速度など、初期値がチューニング済み）
- **ツールパス3Dビューア**（依存ライブラリなしの自前Canvasレンダラ）
  - ドラッグで回転 / ホイールでズーム / Shift+ドラッグでパン
  - レイヤースライダーで層送り・「単層のみ」「移動(travel)表示」「速度で色分け」
- **行番号付き G-code エディタ** — 編集すると自動で3D再描画
- **スタート/エンド G-code 生成** — 左パネルの値から A1 mini 初期化ブロックを生成
- **スニペット挿入** — 一時停止 / 温度変更 / ファン / 色替え(M600) / Zホップ
- **読み込み → 直編集 → ダウンロード** で `.gcode` をそのまま書き出し

## 使い方

1. `index.html` をブラウザで開く
2. 「📂 .gcode 読込」で既存ファイルを開く（または「サンプル生成」）
3. 中央3Dで確認しつつ、右のエディタで直接編集
4. 「⬇ ダウンロード」で保存 → SDカード or Bambu Studio へ

## eSun PLA+ の初期値（目安）

| 項目 | 値 |
|---|---|
| ノズル温度 | 220℃（1層目 225℃） |
| ベッド温度 | 60℃ |
| フロー比 | 0.98 |
| 最大体積速度 | 12 mm³/s |

eSun PLA+ は通常 PLA より粘度が高めなので、温度をやや上げると層間接着が安定します。使用前の乾燥を推奨。
値はあくまで出発点です。各自の環境に合わせて調整してください。

> ⚠️ 純正 Bambu ファーム向けの完全なマクロ展開は行いません。生成される G-code は編集前提の実用テンプレートです。**実機投入前に必ず内容を確認してください。**
