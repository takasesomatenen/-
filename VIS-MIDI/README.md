# VIS-MIDI — Hand Tracking MIDI Controller

Webカメラで映した**手の動き**を MIDI に変換し、**Ableton Live**（や任意の DAW / シンセ）を
ハンズフリーでコントロールする、単一 HTML のアプリです。
手の高さ・位置・つまみ（ピンチ）・開き具合・傾きを MIDI CC / ノートに割り当てられます。

- 依存ライブラリのインストール不要（トラッキングは **MediaPipe Hand Landmarker** を CDN から読込）
- **Web MIDI API** で仮想 MIDI ポートへ直接送信 → Ableton が受信
- 最大 **2 手**・21 ランドマークを検出、左右それぞれ独立に割り当て可能
- 設定はブラウザ内に自動保存

## 動かし方

Web MIDI とカメラは「安全なコンテキスト」でのみ動くため、`file://` で直接開くのではなく
**`localhost` か `https`** で開いてください。

```bash
# このリポジトリのルートで
python3 -m http.server 8000
# → ブラウザで http://localhost:8000/VIS-MIDI/ を開く
```

推奨ブラウザは **Chrome / Edge**（Web MIDI 対応）。初回はカメラ許可のダイアログが出ます。

## Ableton につなぐ

1. **仮想 MIDI ポートを用意**
   - macOS: `Audio MIDI設定 › ウインドウ › MIDIスタジオ › IACドライバ` を開き **「装置はオンライン」** にチェック。
   - Windows: [loopMIDI](https://www.tobias-erichsen.de/software/loopmidi.html)（無料）でポートを 1 つ作成。
2. アプリ右上の **「出力先 MIDI ポート」** でそのポートを選び、**「送信ON」** をチェック（<kbd>Space</kbd> でも切替）。
3. Ableton の `環境設定 › Link/Tempo/MIDI`（旧: Link/MIDI）で、その入力ポートの **「トラック」** と **「リモート」** を ON。
4. Ableton 右上の **MIDIマップモード**（<kbd>⌘/Ctrl+M</kbd>）→ 動かしたいノブ/フェーダーをクリック
   → 実際に手を動かすと、その CC が自動で学習されます。マップモードを解除して演奏開始。

## マッピング

各行は「**どの手** の **どの指標**」を「**CC 番号 / ノート**」と「**MIDI チャンネル**」に割り当てます。

| 指標 | 意味 |
|---|---|
| **X** | 手のひらの横位置 |
| **Y** | 手のひらの高さ（上げるほど大） |
| **Pinch** | 親指と人差し指の距離（つまむ＝小 → 開く＝大） |
| **Spread** | 手の開き具合（グー＝小 / パー＝大） |
| **Tilt** | 手のひらの傾き（ロール角） |

- **CC 行**: 指標（0〜1）を 0〜127 の連続値として送信。`↕` で反転。
- **ノート行**: **Pinch** がしきい値より小さくなった瞬間に Note On、離すと Note Off（つまみでトリガー）。
- 上部の **スムージング** でジッタを抑制、**送信レート上限** で MIDI の流量を制御できます。
- **Panic** ボタンで全ノートオフ＋コントローラリセット。

既定のマッピング例:

| 手 | 指標 | 送信先 |
|---|---|---|
| 右 | Y | CC1 (Mod) |
| 右 | X | CC2 (Breath) |
| 右 | Pinch | CC11 (Expression) |
| 左 | Y | CC74 (Cutoff) |
| 左 | Spread | CC71 (Resonance) |

## プライバシー

映像処理と MIDI 生成は**すべてブラウザ内（端末上）**で完結します。カメラ映像が外部へ送られることはありません。
唯一の外部通信は、初回起動時に MediaPipe のモデル/ランタイムを CDN から読み込む部分だけです。

## 対応状況・既知の制限

- Web MIDI API 対応ブラウザが必要（Chrome/Edge 推奨。Safari は未対応）。
- 仮想 MIDI ポートは OS 側で別途用意が必要（IAC / loopMIDI）。
- 照明・カメラ品質・背景でトラッキング精度が変わります。手が画面内に収まるように。
