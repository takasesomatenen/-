🎛 MIDI 出力 & マッピング

有効 ソース（手 / 指標） 種別 CC/Note Ch ↕
指標（メトリクス）の説明
  • X / Y — 手のひら（第3中手骨）の横・縦位置。上に上げるほど Y=大。
  • Pinch — 親指と人差し指の距離。つまむ＝小 → 開く＝大。
  • Spread — 手の開き具合（グー=小 / パー=大）。
  • Tilt — 手のひらの傾き（ロール角）。
ノート行は Pinch がしきい値より小さくなった瞬間に Note On、離すと Note Off を送ります（つまみでトリガー）。
Ableton へのつなぎ方
  1. 仮想 MIDI ポートを用意
    macOS: Audio MIDI設定 › IACドライバ を有効化。
    Windows: loopMIDI（無料）でポートを1つ作成。
  2. 上の「出力先 MIDI ポート」でそのポートを選び、送信ON。
  3. Ableton の 環境設定 › Link/MIDI で、その入力ポートの 「トラック」「リモート」を ON。
  4. Ableton 右上 MIDIマップモード（⌘/Ctrl+M）→ 動かしたいノブを選択 → 実際に手を動かすと CC が学習されます。
※ Web MIDI と カメラは安全なコンテキストが必要です。 localhost か https で開いてください （ローカルなら例: python3 -m http.server → http://localhost:8000/VIS-MIDI/）。 対応ブラウザは Chrome / Edge を推奨。
設定は自動保存されます（この端末のブラウザ内）。
