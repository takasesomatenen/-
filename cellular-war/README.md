# Cellular War — 赤 vs 青 セルオートマトン対戦

ブラウザだけで動く、ライフゲーム（2色版）を使った対戦ゲームです。単一 HTML・外部通信ゼロ。
`cellular-war/index.html` をブラウザで開くだけで遊べます。

## ルール

- 盤面全体に**標準ライフゲーム**（誕生=近傍3 / 生存=近傍2・3）を適用。
- **誕生するセルの色は、親となった近傍3セルの多数決**（Immigration方式）。
  近傍の3は奇数なので必ず勝敗が付き、境界を押し込むと敵構造を自色に塗り替えられる。
- あなた（**赤**）は盤面の左半分、AI（**青**）は右半分にだけユニットを配置できる。
  攻撃はグライダーや銃を敵陣へ撃ち込んで行う。
- **エナジー**を消費してユニットを配置。大型ユニットほど高コスト＋長い配置クールダウン。
  「でかいものを置くとしばらく置けない／小さいもので効果的に攻める」という駆け引きが生まれる。
- どちらかの色が**全滅したら勝敗確定**（両者全滅は引き分け）。

## 接触戦闘モード（トグル）

デフォルトは純Immigration（色は塗料で、動きは通常ライフ通り＝グライダー衝突による撃ち合い）。
`接触戦闘: ON` にすると、**敵近傍が味方近傍より多いセルは死亡**するルールが加わり、
静止した塊同士が接触面で削り合う近接戦になる。挙動を試して好みで切り替え可能。

## 操作

| 操作 | キー / 入力 |
|---|---|
| ユニット選択 | `1`〜`7` またはパレットのボタン |
| 配置 | 自陣（左半分）にクリック |
| 回転 / 反転 | `R` / `F` |
| 再生・一時停止 | `Space` またはボタン |
| 1手送り | ▷ ボタン |

## ユニット

| ユニット | コスト | 役割 |
|---|---|---|
| 単体セル | 2 | 撹乱・微調整 |
| ブロック | 5 | 静止・素材 |
| 点滅 | 6 | 振動子 |
| グライダー | 12 | 主力の弾（斜めに飛ぶ） |
| 宇宙船(LWSS) | 26 | 直進する重い弾 |
| イーター | 18 | 弾を食う防壁 |
| グライダー銃 | 95 | 連射兵器（高コスト・長クールダウン） |

## 遊び方のコツ

グライダーを列にして撃ち込むと弾幕になる。銃は強力だが本体に敵弾が当たると壊れるハイリスク兵器。
相手の銃の射線にイーターやブロックを置いて弾を止めるなど、構造で受けるのも有効。
