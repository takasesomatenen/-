# GROOVE Generator — ブラウザで動く DNB / ACID / HOUSE ジェネレーター

ブラウザだけで動く、Web Audio API のみで音を生成するグルーヴ・ジェネレーターです。
単一 HTML・外部通信ゼロ・音源ファイル不要。`dnb-generator/index.html` を開くだけで鳴ります。
**DNB（ドラムンベース）・ACID（303風アシッド）・HOUSE（四つ打ち）** の3ジャンルを切り替えて使えます。

## 特長

- **3ジャンル切り替え**（DNB / ACID / HOUSE）— ジャンルタブを押すと次の小節頭でリズム・ベースエンジンごと切り替わる（停止中は即時）。切り替えるとテンポ/スウィング/複雑さ/Hi-Hat密度もそのジャンルの初期値に自動セット
  - **DNB** — ブレイクビート + Reese風デチューン・ベース + サブベース
  - **ACID** — TB-303風モノフォニック・シンセ（スライド／アクセント／レゾナンス・フィルターの"クワッ"を再現）+ 四つ打ちキック
  - **HOUSE** — 四つ打ちキック + オフビートHi-Hat + プラック・ベース
- **10種類のスライダー**（テンポ、スウィング、グルーヴ複雑さ、Hi-Hat密度、ベースのcutoff/レゾナンス/ディストーション、サブベース音量、リバーブ量、マスター音量）が**ジャンル共通・再生を止めずにすべて即座に反映**
- **音色ランダム（🎲 音色ボタン）**を楽器ごとに用意 — Kick / Snare(Clap) / Hats / Bass のそれぞれで、ピッチ・減衰・波形などの音の質感だけをランダムに変更（リズムやノートはそのまま）。Bassの音色ダイスは現在のジャンルのベースエンジン（Reese / 303 / プラック）に応じて変わる
- **表情を変えるボタン**（再生中に押せる）
  - 🎲 パターン変更 / 🎸 ベース変更 — 次の小節頭で滑らかに切り替え（リズム・ノートのランダム化）
  - ⚡ フィル — 小節末にドラムロールを挿入
  - 🌊 ブレイクダウン — ドラムを落としてベースを絞り、リバーブを増やす即時トグル（もう一度押すとドロップ）
  - ⏱ ハーフタイム — キック/セカンダリを半分のノリに切り替え（Hi-Hatはそのまま）
  - Kick / Snare(Clap) / Hats / Bass の個別ミュート
- **キャンバス・ビジュアライザー**（周波数バー + 16ステップのパターングリッド + LEDフラッシュ）で操作の効果を視覚的に確認可能

## 使い方

1. `index.html` をブラウザで開く
2. 上部の「ジャンル」タブで DNB / ACID / HOUSE を選ぶ
3. 「▶ PLAY」を押すと音が始まる（初回クリックで AudioContext を初期化）
4. 再生中にスライダー・ボタン・音色ダイスを触ってリアルタイムに質感・展開を作る

## 仕組み（ざっくり）

- ルックアヘッド方式のスケジューラーで 16分音符グリッドを先読みしながらオシレーター/ノイズをスケジュール
- ベース関連の常時接続ノード（Lowpass Filter → WaveShaper（ディストーション）→ Trim）は3ジャンル共通。DNBのReeseノート・ACIDのモノフォニック・ボイス・HOUSEのプラックノートが同じフィルター/ディストーションへ流れ込むので、cutoff/レゾナンス/ドライブのスライダーはどのジャンルでも同じように"鳴っている音にそのまま効く"
- ACIDはオシレーター1本を常時鳴らし続けるモノフォニック・ボイス方式。ノートごとにフィルターへエンベロープ（アクセントで開き量が変わる）をかけ、スライド・フラグが立っているノート間はオシレーターの周波数だけをポルタメントさせて音を止めずにつなぐ
- リバーブは畳み込み用のインパルス応答をノイズ減衰から自前生成（外部IRファイル不要）
