# DNB Generator — ブラウザで動くドラムンベース・ジェネレーター

ブラウザだけで動く、Web Audio API のみで音を生成する DNB（ドラムンベース）ジェネレーターです。
単一 HTML・外部通信ゼロ・音源ファイル不要。`dnb-generator/index.html` を開くだけで鳴ります。

## 特長

- **ブレイクビート + Reese風ベース + サブベースをリアルタイム生成**（サンプル音源は一切使用しない）
- **10種類のスライダー**（テンポ、スウィング、ブレイク複雑さ、Hi-Hat密度、ベースのcutoff/レゾナンス/ディストーション、サブベース音量、リバーブ量、マスター音量）が**再生を止めずにすべて即座に反映**
- **表情を変えるボタン**（再生中に押せる）
  - 🎲 パターン変更 / 🎸 ベース変更 — 次の小節頭で滑らかに切り替え
  - ⚡ フィル — 小節末にドラムロールを挿入
  - 🌊 ブレイクダウン — ドラムを落としてベースを絞り、リバーブを増やす即時トグル（もう一度押すとドロップ）
  - ⏱ ハーフタイム — キック/スネアを半分のノリに切り替え（Hi-Hatはそのまま）
  - Kick / Snare / Hats / Bass の個別ミュート
- **キャンバス・ビジュアライザー**（周波数バー + 16ステップのパターングリッド + LEDフラッシュ）で操作の効果を視覚的に確認可能

## 使い方

1. `index.html` をブラウザで開く
2. 「▶ PLAY」を押すと音が始まる（初回クリックで AudioContext を初期化）
3. 再生中にスライダーやボタンを触ってリアルタイムに質感・展開を作る

## 仕組み（ざっくり）

- ルックアヘッド方式のスケジューラーで 16分音符グリッドを先読みしながらオシレーター/ノイズをスケジュール
- ベースは 2つのデチューンした Sawtooth + サブSine を、常時接続の Lowpass Filter → WaveShaper（ディストーション）へ通す構成にして、フィルター/ドライブのスライダーが**鳴っている音にそのまま効く**ようにしている
- リバーブは畳み込み用のインパルス応答をノイズ減衰から自前生成（外部IRファイル不要）
